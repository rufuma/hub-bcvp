## week13 使用LoRA训练NER任务

模型结构使用bert模型+crf层+线性层搭建，使用LoRA对少量参数进行微调来完成NER任务。

主要在 q、k、v、o线性层添加LoRA模块，除了LoRA参数、crf层和最后一层线性层，原始模型其余参数均冻结不参与训练
